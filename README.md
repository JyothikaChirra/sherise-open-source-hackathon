# Sherise - Empower & Protect

## Problem Statement
Women often face safety concerns and lack immediate access to emergency assistance.

## Solution
Sherise is a women safety platform that provides emergency support, reporting, community assistance, and safety resources.

## Features
- Emergency SOS
- Incident Reporting
- Community Support
- User Profiles
- Safety Resources

## Tech Stack
- React
- TypeScript
- Vite
- Supabase

## Installation

git clone <repo-url>
npm install
npm run dev
